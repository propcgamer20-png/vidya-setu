Setting up your API key

VidyaSetu runs on GPT-OSS-120B (OpenAI's open-weight model) via Groq's free API — here's what that means and how to connect it.

What is an API key?

An API key is like a personal password that lets an app — in this case, VidyaSetu, running right here in your browser — send requests to a model on your behalf. It's tied to your own Groq account. Nobody else sees it, and VidyaSetu doesn't send it anywhere except directly to Groq.

Why GPT-OSS-120B on Groq?

GPT-OSS-120B is OpenAI's own open-weight model, released under an Apache 2.0 license — anyone can run it, including free API providers. Groq hosts it with a genuinely free tier, no card required, which makes it the easiest way to get VidyaSetu working without spending anything.

How to get a free Groq API key

STEP 1

Go to console.groq.com and sign in (or create a free account if you don't have one).

STEP 2

Find the "API Keys" section in the left-hand menu.

STEP 3

Click "Create API Key", name it if you like, and confirm.

STEP 4

Copy the key right away — it's only ever shown once.

STEP 5

Paste it into VidyaSetu's key field, just under the language buttons.

No billing details needed — Groq's free tier for GPT-OSS-120B costs nothing to start.

⚠ Know your free-tier limits

Groq's free tier for GPT-OSS-120B is generous but not unlimited. As of mid-2026 it's capped at roughly:

  1. 30 requests per minute
  2. 1,000 requests per day
  3. 8,000 tokens per minute
  4. 200,000 tokens per day

That's comfortably enough for normal day-to-day use — just don't loop the app in a script that fires dozens of requests per second, or you'll hit a temporary rate limit (it resets, no charge involved). Check console.groq.com for the current numbers, since providers do adjust free-tier limits over time.

How to add it to VidyaSetu

Once you're in the app, paste your key into the box labelled "Paste your Groq API key" just under the Hindi/Marathi language buttons. That's it — no other setup needed. The key lives only in this browser tab for this session; it isn't saved anywhere when you close the page.

Keep it safe

Treat your key like a password. Don't share this file with your key already pasted in, and don't post screenshots that show it.
